Assistance

Comment pouvons-nous vous aider ?

Snadder est un jeu de serpents et échelles pour deux à quatre joueurs, en local sur un même iPhone ou à distance avec Game Center.

Contacter l’assistance

Décrivez le problème rencontré, le modèle de votre iPhone, votre version d’iOS et la version de Snadder installée. Ne transmettez jamais votre mot de passe Apple ou d’autres informations sensibles.

Écrire à contact@obji.xyz

Questions fréquentes

Faut-il créer un compte Snadder ?

Non. Le mode local ne nécessite aucun compte. Le mode à distance utilise uniquement votre compte Game Center géré par Apple.

Comment jouer à plusieurs sur le même iPhone ?

Sélectionnez « Local », choisissez de deux à quatre joueurs et la difficulté du plateau, puis lancez la partie. Chaque joueur touche son propre dé lorsque son tour arrive.

Comment rejoindre une partie Game Center ?

L’hôte crée un salon et partage son code à six caractères. Les invités sélectionnent « Game Center », saisissent ce code puis attendent que tous les joueurs aient rejoint le salon.

Le code d’invitation ne fonctionne pas

  • Vérifiez chaque caractère du code.
  • Assurez-vous que tous les joueurs utilisent la même version de Snadder.
  • Vérifiez que Game Center est activé et que la connexion internet fonctionne.
  • Si la recherche expire, demandez à l’hôte de créer un nouveau salon.

Game Center ne se connecte pas

Ouvrez les Réglages de l’iPhone, vérifiez votre connexion Game Center, puis relancez Snadder. Une interruption des services Apple peut également empêcher temporairement la création d’un salon.

Compatibilité

Snadder nécessite iOS 17.2 ou une version ultérieure. Une connexion internet et Game Center sont nécessaires uniquement pour les parties à distance.
